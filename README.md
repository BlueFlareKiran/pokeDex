# Pokemon PokeDex

A web app to explore Pokemon details using PokeAPI.

## Features

- View Pokemon info
- See different forms
- Items Discription
- Moves Discription
- Still pending...

## Tech Stack

- React
- Tailwind CSS
- Node.js
- Express
- PokeAPI

## Installation

```bash
npm install
npm run dev
```
